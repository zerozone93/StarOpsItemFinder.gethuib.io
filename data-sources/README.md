# Star Ops Data Source Adapters

Adapters for fetching Star Citizen data from community sources.

| Adapter | Source | Data Type |
|---------|--------|-----------|
| `StarCitizenWikiAdapter` | starcitizen.tools | Weapons, Armor, Resources |
| `UexCorpAdapter` | uexcorp.space | Commodity prices, Terminals |
| `ErkulsAdapter` | erkul.games | Weapon stats, Ship loadouts |
| `FleetViewerAdapter` | fleetyards.net | Ship data |
| `ScTradeToolsAdapter` | sc-trade.tools | Trade routes, Prices |

All adapters currently return stub data.
